Text to speech using pyttsx3 library
before run app

pip install pyttsx3
pip install tk
python tts.py


Working:-
---------------------
1. Enter Any Text or Copypaste context and press speak to check the voice by default voice : male. 
2. For changing voice, select dropdown to female voice
3. adjusting the speed pitch, select range between 100 to 1000 (<210 slower, >210 faster)
4. once the audio is ok, press download to get audio.mp3
